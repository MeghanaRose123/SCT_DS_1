# SCT_DS_1 - Population Distribution Bar Chart

## Task
Visualize the distribution of a continuous variable using a bar chart
— showing world population distribution by country.

## Dataset
- Source: World Bank
- Indicator: Total Population (SP.POP.TOTL)
- Year Used: 2023

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- NumPy

## Output
Horizontal bar chart showing the **Top 15 Most Populous Countries in 2023**
